# StudyOS AI

## Objetivo

O StudyOS AI é uma plataforma para gerenciamento da evolução profissional.

O sistema controla:

- Trilhas de estudo
- Projetos
- Certificações
- Agenda
- Horas estudadas
- IA Mentora
- Cronograma

---

- SQLite

- Plotly
